# UI Pages for Evaluation

Based on the navigation menu, the following pages are available for UX evaluation:

## Main Pages

1. **Chat** (`/chat`) - Main chat interface with conversation sidebar
2. **History** (`/history`) - Chat history
3. **Help** (`/docs/`) - Help documentation

## Data Management

4. **Notes** (`/notes`) - Note management interface
5. **Context** (`/context`) - Context management

## Document Management

6. **Documents List** (`/documents/`) - List of uploaded documents
7. **Documents Upload** (`/documents/upload`) - Document upload interface
8. **Documents Search** (`/vector-search`) - Document search with vector embeddings

## Automation

09. **Events** (`/events`) - Event management
10. **Event Listeners** (`/event-listeners`) - Event listener configuration

## Internal/Admin

11. **Tools** (`/tools`) - Available tools interface
12. **Task Queue** (`/tasks`) - Background task queue status
13. **Error Logs** (`/errors`) - System error logs
